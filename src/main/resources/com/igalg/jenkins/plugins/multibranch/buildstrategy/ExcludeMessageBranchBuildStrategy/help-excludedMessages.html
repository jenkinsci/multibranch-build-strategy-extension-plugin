Each exclusion uses Java pattern matching based on standard java.util.regex package, and must be separated by a new line.
An empty list implies that nothing is excluded. 
.*\[ci\-skip\].*
.*\[maven\-release\-plugin\].*
The example above illustrates that if last commit contains "[ci-skip]" or "[maven-release-plugin]" phrase somewhere in commit message, a build will not occur.
